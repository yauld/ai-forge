# RAG 实战

这个目录记录 RAG 从概念到最小问答链路的实践路径：文档加载、文本切分、向量化、向量库、Retriever 和 Prompt 组装。

## 推荐路线

| 阶段 | 内容 | 文件范围 |
| --- | --- | --- |
| 基础概念 | RAG 是什么、向量是什么、余弦相似度怎么算 | `01` - `03` |
| 文档处理 | CSV、JSON、PDF、Text、Markdown、目录加载 | `04` - `06` |
| 切分与索引 | Text Splitter、向量库增删查与持久化 | `07` - `08` |
| 检索与问答 | Retriever、Prompt 组装、LLM 调用 | `09` - `10` |

## 重点主题

- RAG 的难点不只是“向量库”，还包括文档结构、切分策略和检索质量。
- Retriever 决定了模型能看到什么上下文。
- 最小问答链路应该先跑通，再逐步引入重排、评估和更复杂的数据治理。

## 适合读者

- 想从零理解 RAG 工程链路的人。
- 正在把业务文档接入大模型的人。
- 想建立向量检索直觉的人。
