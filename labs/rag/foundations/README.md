# RAG 实战专题

这个专题从 RAG 的基本问题出发，依次研究向量、文档加载、文本切分、向量库、Retriever 和最小问答链路。

重点不是快速拼出一个 Demo，而是理解数据怎样被处理、检索器怎样决定模型能看到什么，以及后续应该如何评估和优化检索质量。

## 你会学到什么

- RAG 的完整工程链路和适用边界。
- 文本向量与余弦相似度的基本直觉。
- 常见文档格式如何加载和切分。
- 向量库与 Retriever 如何组织检索。
- 如何把检索结果、Prompt 和模型组成最小问答链路。

## 适合读者

- 想从零理解 RAG 工程链路的人。
- 正在把业务文档接入大模型的人。
- 想进一步研究召回质量和答案可信度的开发者。
