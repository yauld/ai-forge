# 周报生成流程

使用这份 Skill 帮用户整理一份结构清晰、信息克制的工作周报。

执行步骤：

1. 读取本周输入材料，提取完成事项、未完成事项、风险和下周计划。
2. 如需模板，只能参考当前 Skill 自己的 `references/weekly_report_template.md`。
3. 不读取薪酬、绩效、私人备忘录或其他 Skill 的内部资料。
4. 输出周报草稿，并标注需要用户补充的信息。
