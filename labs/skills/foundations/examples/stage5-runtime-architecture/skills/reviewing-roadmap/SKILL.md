# 路线图审查

使用这个 Skill，把路线图当作学习计划或执行计划来审查，而不是只润色标题列表。

## 执行步骤

1. 判断路线图是否有清晰目标和目标读者。
2. 检查阶段顺序是否符合认知递进，是否存在重复阶段。
3. 检查每个阶段是否有明确研究问题、实验内容和产出物。
4. 调用 `text_stats` 观察路线图文本规模，辅助判断内容是否过长或过碎。
5. 输出审查摘要，优先指出结构性问题，再给出可执行修改建议。
