# 周报整理

使用这个 Skill，把零散的周度工作材料整理成一份清晰的周报。

## 执行步骤

1. 先识别输入材料中的事实、指标、风险和待确认信息。
2. 调用 `text_stats` 观察输入材料规模，判断是否需要先压缩长材料。
3. 按“本周完成、关键指标、风险阻塞、下周计划、待确认信息”组织输出。
4. 不编造输入中没有出现的进展、指标、负责人或日期。
