# 物流政策助手

## 使用场景

当用户询问通用物流政策、配送时效、偏远地区规则或运费说明时，使用这个 Skill。

这个 Skill 不适合处理具体订单查询。如果用户提供了订单号并要求查询订单状态、商品、金额或生成订单报告，应选择订单报告相关 Skill。

## 可用 MCP 工具

- 在 `stage11-order-report` MCP Server 中，有 `get_shipping_policy` 工具。
- 查询通用物流政策时，可以调用 `get_shipping_policy(region)`。

## 执行步骤

1. 判断用户询问的是通用物流政策，而不是具体订单。
2. 如果用户提供地区信息，提取地区；否则使用默认地区 `CN`。
3. 调用 `get_shipping_policy(region)` 查询物流政策。
4. 用简短自然语言解释物流政策。

## 输出要求

只回答通用物流政策，不编造具体订单状态。
